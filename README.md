# 🌍 Travel Explorer ✈️

Travel Explorer is a simple and interactive web application that helps users **plan trips to their dream destinations**.  
Users can enter a destination, choose travel duration and travel type, and the application will provide:

- 🌤️ Current weather information of the destination
- 🖼️ Beautiful destination image
- 🚆 Travel suggestions
- 💡 Personalized travel tips

The application integrates **OpenWeather API** and **Unsplash API** to fetch real-time data.

---

# 🚀 Features

- 🔍 Search for any travel destination
- 🌤️ Get **real-time weather information**
- 🖼️ Display **random destination images** from Unsplash
- ✈️ Travel recommendations based on selected destination
- 💡 Personalized travel tips depending on travel type:
  - Solo
  - Family
  - Friends
  - Romantic
- ⏳ Loading animation while fetching data
- 📱 Responsive design

---

# 🛠️ Tech Stack

- **HTML5** – Structure of the website
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Application logic and API calls
- **OpenWeather API** – Weather data
- **Unsplash API** – Destination images

--

# ⚙️ How It Works

1. User enters a **destination**.
2. User selects:
   - Trip duration
   - Travel type
3. The app fetches:
   - Weather data from **OpenWeather API**
   - Destination image from **Unsplash API**
4. The system analyzes the weather and generates **travel tips**.
5. The results are displayed on the screen.

---

# 🔑 APIs Used

## OpenWeather API
Used to fetch the **current weather and temperature** of the destination.

https://openweathermap.org/api

## Unsplash API
Used to fetch **random high-quality images** of the destination.

https://unsplash.com/developers
